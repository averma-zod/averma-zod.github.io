Project Overview

This project involves the deployment of a Node.js web application using Docker containers and a CI/CD pipeline managed by Jenkins. The project emphasizes security throughout the development and deployment lifecycle.

Project Description

This project employs Docker containers with Nginx reverse proxy for scalable deployment. Jenkins orchestrates CI/CD pipelines integrating security plugins like OWASP Dependency Check, SonarQube, Trivy, Prometheus, and Grafana for vulnerability detection and monitoring. AWS security configurations, including Security Groups, Elastic IPs, and VPCs, helped ensure higher infrastructure security. Utilizing Prometheus and Grafana, real-time metrics and logs aid in proactive security monitoring. Pre and post-build security checks ensure early vulnerability mitigation, enhancing overall system resilience and integrity.
